Xojo Syntax Highlight

Syntax highlighting for the Xojo programming language — four library implementations with One Dark / One Light color schemes.

highlight.js

JavaScript

ES module factory function. Register once, works with any highlight.js theme. Ideal for static sites and Markdown renderers.

Prism.js

JavaScript

Self-executing IIFE that registers with Prism automatically. Load after prism.js and use language-xojo on any block.

CodeMirror 6

JavaScript

StreamParser for CodeMirror 6. Fully editable code editor with Lezer highlight tags and One Dark / default light theme.

Pygments

Python

RegexLexer for server-side rendering with Python. Includes XojoOneDarkStyle and XojoOneLightStyle — no CDN dependencies.
